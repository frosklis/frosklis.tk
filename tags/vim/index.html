Usando vim

 2019-07-02
  vim, hugo

El ratón es lo más cómodo pero el teclado es lo más rápido. Por eso quiero usar vim más en la medida de lo posible. Por ejemplo para bloguear. En Hugo se puede crear un post con el comando hugo new posts/un-post-nuevo.md y, para hacerlo más cómodo se puede añadir la línea NewContentEditor = "vim" al config.toml de modo que al escribir ese comando se abra vim.
